PhD

I will shortly begin a PhD at York University.

During my PhD, I will be studying the effect of environmental noise on sleep quality. My aim is to create a definition of disturbing noise, including the creation of a system that can identify not only if a noise is disturbing, but the effect that nosie is likely to have on nearby sleepers.

The long-term goal of this work is to create a small system that can inform users of the quality of their sleep, and can provide non-invasive masking noise that will improve the sleep quality of people in a range of environments such as intensive care and urban centers.

The typical custom DSP block development process is as follows:

  • Definition of the desired algorithm
  • Prototype of algorithm in Matlab
  • Benchmarking of algorithm using Matlab in DSP native numerical format (such as Fixed Point for AD Sigma DSPs)
  • Algorithm implementation in native environment
  • Implementation testing against Benchmark
  • Further user interface development (if necesarry)
  • Project conclusion (excluding any support agreement)
